# Frontend - Coding Champion

React + Vite frontend for Coding Champion.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

App runs on `http://localhost:5173`

## Testing

```bash
# Run tests
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with coverage
npm run test:coverage

# Run tests with UI
npm run test:ui
```

## Project Structure

```
frontend/
├── src/
│   ├── components/    # React components
│   ├── test/          # Test setup files
│   └── ...
├── public/            # Static assets
└── vitest.config.js   # Vitest configuration
```

## Tech Stack

- React 19
- Vite
- Vitest (testing)
- React Testing Library
