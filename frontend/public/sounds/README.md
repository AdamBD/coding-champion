# Final Fantasy Victory Fanfare Sound

Place the **classic Final Fantasy Victory Fanfare** sound file here.

This is the iconic sound that plays when you:

- Win a battle in Final Fantasy
- Level up in Final Fantasy
- Complete quests in Final Fantasy games

## File Name

Name the file: `ff-victory.mp3` (or `.wav`, `.ogg`)

## Where to Get the Sound

Download the classic Final Fantasy Victory Fanfare from:

- **MyInstants**:
  https://www.myinstants.com/en/instant/final-fantasy-victory-fanfare/
  - Click the download button to get the MP3 file
- **FreeSound.org**: Search for "Final Fantasy victory fanfare"
- **YouTube**: Search for "Final Fantasy victory fanfare" and use a YouTube to
  MP3 converter

## Supported Formats

- MP3 (recommended)
- WAV
- OGG

## Installation

1. Download the sound file
2. Rename it to `ff-victory.mp3`
3. Place it in this directory: `frontend/public/sounds/ff-victory.mp3`
4. The sound will automatically play when quest steps are completed!

**Note**: Make sure you have the rights to use the sound file in your project.
